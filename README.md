# Python controlled Turing smart screen - Linux only

Original source for this fork - https://github.com/mathoudebine/turing-smart-screen-python

The complete Windows and Mac OS support was removed and GPUtils were replaced by pynvml

The project is primarily developed for TURZX 3.5" screen as this is the only screen available to me.
